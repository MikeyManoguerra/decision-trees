### Tech Stack Frontend
- React/Redux
- Redux-Form
- d3 Cytoscape for grapchical interface
- Enzyme/Jest
- Deployed via Netlify

### Tech Stack Backend
- Node.JS/Express
- JWT with Passport
- MongoDB/Mongoose
- Database hosted on mLab
- Chai/Mocha
- Deployed via heroku
